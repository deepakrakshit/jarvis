# AI Assistance Statement

This project uses AI assistance to accelerate development tasks such as:

- code drafting and refactoring
- boilerplate generation
- documentation scaffolding
- repetitive implementation work

## Human-Controlled Responsibilities

All critical engineering accountability remains human-controlled, including:

- testing strategy and execution decisions
- debugging and root-cause validation
- architecture direction and tradeoff decisions
- security, privacy, and compliance decisions
- production readiness and release approval

## Review Policy

AI-generated changes are treated as proposed code and are reviewed before being accepted.
No AI output is assumed correct without validation.

## Production Policy

Final production decisions are made by human maintainers.
AI assistance is a productivity tool, not an autonomous decision-maker.
